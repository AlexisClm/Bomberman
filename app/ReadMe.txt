Date : 08/03/2020

Auteur : CLEMENT Alexis

Description : Jeu d'action/labyrinthe (inspiré du jeu "Bomberman") où le but de la partie en mode multi, est de de faire exploser les adversaires, ou en mode aventure, de trouver la sortie.


Contrôles :
	- Menu : Clic gauche = Accès au jeu en mode solo
	                     = Accès au jeu en mode duo
		      	     = Quitter le jeu
		 Echap = Quitter le jeu

	- Jeu : 
		- Bomber blanc : 
			Z = Déplacer vers le haut
			S = Déplacer vers le bas
			Q = Déplacer vers la gauche
			D = Déplacer vers la droite
			Espace = Poser une bombe
			Echap = Accès aux options

		- Bomber noir : 
			Flèche haut = Déplacer vers le haut
			Flèche bas = Déplacer vers le bas
			Flèche gauche = Déplacer vers la gauche
			Flèche droite = Déplacer vers la droite
			Entrer = Poser une bombe

	- Options : Clic gauche = Retour au jeu
		    	        = Retour au menu
		    	        = Quitter le jeu
		    Echap = Retour au jeu

	- GameWin : Clic gauche = Rejouer une partie
		    	        = Retour au menu
		    Echap = Retour au menu

	- GameOver : Clic gauche = Retour au jeu
		    	         = Retour au menu
		     Echap = Retour au menu

Bugs connus :
	- L'animation des caisses qui se détruisent peut se reproduire alors qu'elles se sont déjà détruites avant
	- Les sons du jeu ne se mettent pas en pause dans le mode "pause"

Fonctionnalités du jeu : Pour gagner, le joueur doit :
	- En solo :
		- Ne pas se faire toucher par l'explosion des bombes (ne pas tomber à 0 HP)
		- Trouver la clé cachée dans une des caisses pour accéder à la porte
	- En duo :
		- Ne pas se faire toucher par l'explosion des bombes (ne pas tomber à 0 HP)
		- Toucher l'autre joueur avec les bombes
